# Covid19 Data Analysis and Prediction

## Project Overview
This project analyzes COVID-19 data using R. It includes data cleaning, exploratory data analysis (EDA), visualization, and basic predictive modeling to understand trends and patterns of the pandemic.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization using charts and graphs
- Trend analysis of COVID-19 cases
- Basic predictive modeling

## Technologies Used
- R
- RStudio
- Libraries: ggplot2, dplyr, tidyverse

## Project Structure
- R scripts: Contains all analysis code
- Data: Dataset used for analysis
- README.md: Project documentation

## How to Run
1. Open the project in RStudio
2. Install required libraries (ggplot2, dplyr, tidyverse)
3. Run the R script file

## Results
This project provides insights into COVID-19 trends, including case growth patterns and predictions.
